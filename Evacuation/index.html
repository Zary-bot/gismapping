Map Information

This interactive map uses the QGIS BaseMap.qgz OpenStreetMap source as the Leaflet basemap. It shows evacuation centers and related facilities around Tunga. Use search, layer control, zoom, and marker popups to explore each location.

Selected Location

Click a marker on the map or choose a search result to view location name, description, image, type, capacity, and contact details.

Features

  • Interactive MapPan, zoom, and explore the map.
  • Search LocationType a place name, press Enter, or click the search button to zoom to a location.
  • Layer ControlShow or hide evacuation centers, halls, schools, and courts.
  • Marker PopupsOpen detailed information cards.
  • LegendIdentify symbols and colors.
  • Responsive DesignWorks on desktop and mobile screens.
